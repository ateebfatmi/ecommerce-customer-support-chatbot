# AI-Powered E-commerce Customer Support Chatbot

An AI-powered customer support chatbot built using **Python, Flask, Machine Learning, HTML/CSS, and MySQL** to automate customer query handling for e-commerce platforms.

---

# Features

* AI-based intent classification using Machine Learning
* Customer support query handling
* Order tracking functionality
* Refund and cancellation query support
* Complaint management system
* Chat history storage
* Responsive web interface
* Flask backend integration
* MySQL database integration
* Deployable on Render

1. Dataset preparation
2. Text vectorization using CountVectorizer
